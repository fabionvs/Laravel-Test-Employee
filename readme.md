## - Installation

First, use composer to install Laravel dependencies:
```
php composer install
```

Configure database inside .env file:
```
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=test
DB_USERNAME=posgres
DB_PASSWORD=secret
```
